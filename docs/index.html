json‑seal

Lightweight, portable JSON data integrity — without the complexity.

import { seal, verify } from "json-seal";

const sealed = await seal({ message: "hello" });
const result = await verify(sealed);

json‑seal creates a cryptographically signed, tamper‑evident envelope around any JSON‑serializable value. It uses the RFC 8785 JSON Canonicalization Scheme and RSA‑PSS signatures to guarantee both integrity and authenticity.
